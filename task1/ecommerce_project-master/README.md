# 🛒 Simple E-commerce Store

This is a full-stack e-commerce web application developed using Django (Python) for the backend and HTML, CSS, and JavaScript for the frontend.

The project demonstrates core e-commerce functionalities including product management, shopping cart operations, and order processing, making it a solid foundation for scalable online store systems.

---

## 🚀 Features

- 🛍️ Product listing and details page
- 🛒 Shopping cart functionality
- 📦 Order processing system
- 👤 User registration and authentication
- 🔗 REST API integration (Django REST Framework)
- 📊 Dynamic frontend with JavaScript

---

## 🧰 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Backend
- Django
- Django REST Framework

### Database
- SQLite (default, can be upgraded to PostgreSQL)
